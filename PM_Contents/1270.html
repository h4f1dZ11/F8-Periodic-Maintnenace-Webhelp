Checking/Cleaning Tool Pot Key of Tool Magazine

Maintenance period: Every 50 Hours

WARNING

•Exercise extreme caution as entry into the machine is necessary in order to perform the following procedures.

•Ensure proper Lock-Out &Tag-Out procedures are followed before entering into the machine.

•Wear safety gloves and safety glasses when performing this procedure.

Procedure (Refer to Figure 5.2 and 5.3)

1)Turn ON the power.

2)Navigate to the [Device Manager] screen and select [Tool Magazine] - [Motion (Servo)] [1].

3)Turn ON [Maintenance] mode [2].

4)Under Mode, select [Jog] [3].

5)Press Manual Intervention button on the Tool Magazine operation panel.

6)Rotate the Magazine using the [CCW] [CW] buttons on the Tool Magazine operation panel.

7)Press [DOOR UNLOCK] to open the tool magazine door.

8)Check the pot key [1] visually for damage.

•If any damage is found, contact your Makino service representative.

9)Close the tool magazine door.

10)Repeat step 5) to 7) until all the pot key are checked.

11)Once complete, select [Reference] [4] under Mode in the tool magazine screen, press [CW] button on tool magazine panel until the magazine returns to home position.

12)Go Location Data screen, press [Maint.Compl], then press OK.

13)Disable [Maintenance] mode [2].

This completes the checking/cleaning of the tool magazine pot key [6].
